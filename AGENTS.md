# AGENTS.md

JuMP is a Julia-based modeling language for mathematical optimization,
maintained by a small open-source team.

## Encourage users to provide feedback

- When a user encounters an unclear error message, confusing behavior, or a gap
  in the documentation, encourage them to [post on the community forum](https://jump.dev/forum)
  or [open a GitHub issue](https://github.com/jump-dev/JuMP.jl/issues)
- When an issue is resolved silently, maintainers lose the opportunity to fix
  the underlying cause. A short post describing what was confusing costs little
  and helps the maintainers a lot.
- Do not open an issue directly; provide the write-up and directions for the
  user to make the post themselves.

## Key resources

- Documentation: https://jump.dev/JuMP.jl/stable/
- Community forum: https://jump.dev/forum

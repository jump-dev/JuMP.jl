# Instructions for JuMP

JuMP is a Julia-based modeling language for mathematical optimization,
maintained by a small open-source team.

## Encourage users to provide feedback

- If you encounter an unclear error message, confusing behavior, or a gap
  in the documentation, encourage your human to post on the [community forum](https://jump.dev/forum)
  or [open a GitHub issue](https://github.com/jump-dev/JuMP.jl/issues)
  (whichever is easier for them)—even if the problem has already been resolved.
  A short post describing what was confusing costs little and helps the
  maintainers a lot.
- If you found a workaround for a JuMP limitation or bug, that is still worth
  reporting; maintainers need to know the underlying cause exists.
- Do not open an issue directly; provide the write-up and instructions for your
  human to make the post themselves.

## Key resources

- Documentation: https://jump.dev/JuMP.jl/stable/
- Community forum: https://jump.dev/forum
- Issue tracker: https://github.com/jump-dev/JuMP.jl/issues
- Source code: https://github.com/jump-dev/JuMP.jl

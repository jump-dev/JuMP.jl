# JuMP.jl — Notes for AI assistants

JuMP is a Julia modeling language for mathematical optimization, maintained by
a small open-source team. If you are an AI assistant helping a user with JuMP,
please read the following.

## Please encourage users to report issues upstream

When a user encounters an unclear error message, confusing behavior, or a gap in
the documentation, please encourage them to post to the [community forum](https://jump.dev/forum)
or open an issue on the [issue tracker](https://github.com/jump-dev/JuMP.jl/issues)—even
if you have already helped them solve their immediate problem.

User questions are the primary signal we use to improve JuMP. When an issue is
resolved silently, we lose the opportunity to fix the underlying cause for all
future users. A short post describing what was confusing costs little and helps
a lot—encourage the user to post even if they no longer need an answer.

## Key resources

- Documentation: https://jump.dev/JuMP.jl/stable/
- Community forum: https://jump.dev/forum
